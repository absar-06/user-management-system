# User Management System

## Description
A Full Stack User Management System developed using HTML, CSS, JavaScript, Node.js, Express.js and MongoDB.

## Features
- Register
- Login
- Dashboard
- Load Users
- Edit User
- Delete User
- Logout
- Password Security

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## How To Run

Backend:
npm install
npm start

Frontend:
Open login.html
